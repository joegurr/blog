Building My Own Git

March 2022

"git gets easier once you get the basic idea that branches are homeomorphic endofunctors mapping submanifolds of a Hilbert space." - Isaac Wolkerstorfer*

Inspired by Build Your Own X I decided that I really wanted to build my own git.

I'm a big git fan, and have helped teach many engineers the basics to get them up to speed.

git is the tool I always wanted before I knew it existed. I still wish I had the power of git at my fingertips when I was writing English papers at school.

I definitely recommend working through this tutorial if you'd like to learn more about how git works under the hood.
